# MSOC
Данный проект создан для быстрого поиска и скачивания музыки без ограничений!<br>
Вам лиш стоит ввести название музыки, как вам выведутся возможные варианты песен, а так же ссылки для их скачивания<br>

**cli** - консольная версия MSOC<br>
**gui** - версия MSOC с графическим интерфейсом<br>
**site** - версия MSOC в виде полноценного сайта<br>
