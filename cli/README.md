# MSOC (Music Search On Console)
Данная консольная утилита создана для быстрого поиска и скачивания музыки без ограничений!<br>
Вам лиш стоит ввести название музыки, как вам выведутся возможные варианты песен, а так же ссылки для их скачивания<br>
Так же присутствует функция прослушивания найденных песен на локальном сайте
