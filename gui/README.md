# MSOG (Music Search On GUI)
Данная программа для андроид создана для быстрого поиска и скачивания музыки без ограничений!<br>
Вам лиш стоит ввести название музыки, как вам выведутся возможные варианты песен, а так же ссылки для их скачивания<br>
